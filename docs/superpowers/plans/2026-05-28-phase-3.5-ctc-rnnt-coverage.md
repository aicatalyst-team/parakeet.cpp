## Task 2: RNNT greedy loop + routing + validate parakeet-rnnt-*

**Files:** Create `src/rnnt.hpp`, `src/rnnt.cpp` (or add `rnnt_greedy` to `src/tdt.cpp`); modify `src/parakeet.cpp` (routing); modify `docs/parity.md`.

**RNNT greedy (port `GreedyRNNTInfer._greedy_decode`):**
```
t=0; hyp=[]; state=zero_state; have_token=false; last=SOS
while t < T:
    emitted=0
    while emitted < max_symbols:
        is_sos = !have_token
        pred.step(last, is_sos, state, g, newstate)
        joint(enc[t], g) -> logits[vocab+1]      # RNNT joint: NO duration head, V_plus=vocab+1
        k = argmax(logits)
        if k == blank: break                      # blank -> advance time
        hyp.push(k); last=k; have_token=true; state=newstate; emitted++   # commit on non-blank, stay at t
    t += 1
```
Key differences from TDT: no duration logits (the RNNT `Joint` output is `vocab+1`; `num_durations()==0`); on blank you advance `t` by exactly 1 (no skip); on non-blank you emit and stay at the same frame (capped by `max_symbols`). `max_symbols=10` (NeMo default; same as TDT).

- [ ] **Step 1: Interface** `std::vector<int32_t> pk::rnnt_greedy(const PredictionNet& pred, const Joint& joint, const std::vector<float>& enc, int T, int enc_hidden, int blank_id, int max_symbols);`
